อธิบายการทำงานของโค้ด

โปรแกรมนี้เป็น **ELIZA Chatbot** อย่างง่าย ทำงานบน Terminal โดยใช้หลักการ **Pattern Matching (จับคู่ข้อความ)** ร่วมกับ **Regular Expression (โมดูล `re`)** เพื่อโต้ตอบกับผู้ใช้เลียนแบบนักจิตวิทยา มีโครงสร้างหลัก 2 ส่วนดังนี้:

1. ฟังก์ชัน `eliza_response(user_input)`

ทำหน้าที่วิเคราะห์ข้อความที่ผู้ใช้พิมพ์เข้ามา โดยจะไล่เช็กกับ **"กฎการตอบกลับ" (Rules)** จากบนลงล่างทีละข้อ:

- ดักจับคำสำคัญ (Keyword Detection):** มองหาแพทเทิร์นคำ เช่น `I'M SAD`, `ALL`, `ALWAYS`, `MY BOYFRIEND/GIRLFRIEND` (ไม่สนใจตัวพิมพ์เล็ก-ใหญ่)

- ดึงคำมาตอบกลับ (Dynamic Response):** หากตรงเงื่อนไข `I'M ...` โปรแกรมจะดึงข้อความข้างหลังคำนั้น มาแปลงเป็นตัวพิมพ์ใหญ่แล้วโยนกลับไปเป็นคำถามย้อนผู้ใช้

- แผนสำรอง (Fallback):** หากพิมพ์ข้อความที่ไม่ตรงกับกฎข้อไหนเลย โปรแกรมจะตอบประโยคกลาง ๆ ว่า `"PLEASE TELL ME MORE."` เพื่อชวนคุยต่อ

2. ฟังก์ชัน `main()`

ทำหน้าที่ควบคุมหน้าจอและการรับส่งข้อมูล:

- ใช้ลูปไม่รู้จบ (`while True:`) เพื่อรอรับคำสั่งจากผู้ใช้ ให้สามารถคุยโต้ตอบได้เรื่อย ๆ

- มีระบบดักจับคำว่า **`quite`** เพื่อใช้สำหรับพิมพ์สั่งปิดโปรแกรมและออกจากลูป
